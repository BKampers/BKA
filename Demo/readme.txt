To run the calendar demo:
mvn exec:java -Djava.util.logging.config.file="calendar-logging.properties"  -Dexec.mainClass="bka.demo.calendar.CalendarDemo"

To run the Weather station demo:
mvn exec:java -Djava.util.logging.config.file="weatherstation-logging.properties" -Dexec.mainClass="bka.demo.clock.weatherstation.WeatherStationDemo"
